# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately.

- **Do not** open a public issue for security problems.
- Email the maintainer at: info@osswishlist.org
- Include as much detail as possible so we can quickly reproduce and address the issue.

We will acknowledge your report within 1 business days and provide updates as we investigate and resolve the issue.


Thank you for helping keep OSS Wishlist and its users safe!
